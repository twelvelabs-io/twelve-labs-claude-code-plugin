# Plugin Marketplace Research

## Goal

Make `twelvelabs-io/twelve-labs-claude-code-plugin` installable via:

```
/plugin marketplace add twelvelabs-io/twelve-labs-claude-code-plugin
/plugin install twelvelabs@twelvelabs-plugins
```

## Task

Find real, working Claude Code plugin marketplaces on GitHub. Look at their actual `marketplace.json` files and repo structure to determine the standard format that works for end users out of the box.

## Questions to Answer

- What does `marketplace.json` look like in real published plugins?
- What `source` format do they use?
- What does the repo directory structure look like?
- Do they use `plugin.json` + `marketplace.json` or just one?
- How do they handle the plugin being in the same repo as the marketplace?
